and one library file for each paper, currently:
  * zhang2007.py
  * zhang2009.py
  * zhang2010.py
  * genot2011.py
  * kotani2017.py
  ...
soon:
  * yin2008.py
  * zhang2010cooperative.py
  * qian2011.py
  * groves2015.py
  * srinivas2017.py
  * seelig2006.py

In order to (re)produce results, you might want to adapt
settings in the library files.
