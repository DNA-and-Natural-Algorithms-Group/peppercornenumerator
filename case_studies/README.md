This directory contains an executable python script:
  * figure_analysis.py

and one library file for each paper, currently:
  * zhang2007.py
  * zhang2009.py
  * zhang2010.py
  * zhang2010cooperative.py
  * genot2011.py
  * qian2011.py
  * kotani2017.py
  * zhang2009_rates.py
  * dabby2013_rates.py

  ...

In order to (re)produce results, you might want to adapt settings in the
library files.
